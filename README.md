# To-Do List App (Java)

## Project Overview
The To-Do List App is a console-based application developed using Java.  
It helps users manage daily tasks by allowing them to add, view, complete, and delete tasks.

This project is useful for beginners to understand Java programming concepts such as:
- Classes and Objects
- ArrayList
- Scanner
- Loops
- Conditional Statements
- Object-Oriented Programming (OOP)

---

## Features
✅ Add new tasks  
✅ View all tasks  
✅ Mark tasks as completed  
✅ Delete tasks  
✅ Exit application  

---

## Technologies Used
- Java
- ArrayList
- Scanner Class
- OOP Concepts

## How to Run
1. Install Java (JDK 8 or above)  
2. Open terminal / command prompt  
3. Compile the files:

```bash
javac TodoListApp.java
```

4. Run the program:

```bash
java TodoListApp
```
